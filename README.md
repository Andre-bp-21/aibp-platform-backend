# AIBP Platform - Backend API

API profesional de la Plataforma Adaptativa de Inteligencia Empresarial.

## 🚀 Características

- Autenticación con JWT
- Análisis empresarial avanzado
- Simulación de escenarios
- Procesamiento de archivos Excel/CSV
- Base de datos MongoDB
- API RESTful completa

## 🛠️ Tecnologías

- Node.js + Express
- TypeScript
- MongoDB + Mongoose
- JWT Authentication
- Multer para uploads

## 📋 Requisitos

- Node.js 18+
- npm o yarn
- MongoDB Atlas (gratis)

## 🏃 Inicio Rápido

```bash
# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Compilar para producción
npm run build

# Iniciar en producción
npm start
